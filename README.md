Instructions for using Gator:

REQUIREMENTS:
You will need both Go and Postgres installed on your machine

1: Within your CLI, run "go install https://github.com/Jon-Castro856/Gator" to install gator onto your machine

2: created a file called ".gatorconfig" within your user folder (home/*usernamehere*)

3: within your file, add the following information:

{
 "db_url": "postgres://postgres:sql@localhost:5432/gator?sslmode=disable",
 "current_user_name": "jon"
}

4: figure out the rest of the program
